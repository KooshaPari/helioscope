# Specifications

## Acceptance Criteria

- README badge block includes the Sladge badge.
- Badge links to `https://sladge.net`.
- No runtime, dependency, lockfile, MCP, or Codex code changes are included.
- Session docs record why an isolated worktree was used.

## Out Of Scope

- Runtime behavior changes.
- MCP server changes.
- Codex fork synchronization.
- Lockfile regeneration.
